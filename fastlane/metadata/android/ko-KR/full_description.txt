Tikka Timer는 알람시계, 타이머, 스톱워치를 하나의 아름다운 앱에 담은 심플하면서도 강력한 시간 관리 앱입니다.

<b>⏰ 알람</b>
• 라벨을 지정하여 여러 개의 알람 생성
• 요일별 반복 알람 설정
• 스누즈 기능 지원
• 다양한 알람음 선택
• 진동 지원

<b>⏱️ 타이머</b>
• 시, 분, 초 단위로 타이머 설정
• 자주 사용하는 타이머를 프리셋으로 저장
• +1분 빠른 추가 버튼
• 아름다운 원형 진행 표시기
• 백그라운드 실행 지원

<b>⏲️ 스톱워치</b>
• 밀리초 단위 정밀 측정
• 무제한 랩 타임 기록
• 최고/최저 랩 타임 자동 하이라이트
• 깔끔하고 직관적인 인터페이스

<b>🎨 맞춤 설정</b>
• 6가지 아름다운 컬러 테마
• 라이트/다크 모드 지원
• 시스템 테마 자동 적용
• 다국어 지원 (한국어, 영어, 일본어, 중국어)

<b>🔒 개인정보 보호</b>
• 인터넷 권한 불필요
• 모든 데이터는 기기에 로컬 저장
• 광고 없음, 추적 없음
